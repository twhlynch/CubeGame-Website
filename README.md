# CubeGame LAN Viewer

Web-based viewer for CubeGame, built with Three.js, TypeScript, and Vite.

## Quick start

```bash
npm install
npm run dev
```

## Commands

| Command           | Description      |
| ----------------- | ---------------- |
| `npm run dev`     | Start dev server |
| `npm run build`   | Build to `dist/` |
| `npm run preview` | Preview build    |

## Usage

Enter a CubeGame server IP address and click **Connect** to view the game state
in 3D.
